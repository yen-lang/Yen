---

## Security Priorities

- Prevent remote code execution vulnerabilities.
- Prevent memory corruption, buffer overflows, or crashes (especially after multi-platform support).
- Prevent privilege escalation through language features.
- Maintain a secure sandbox in future runtime features (planned).

---

## In Progress

- Hardened input validation at parsing and runtime.
- Future sandbox mode for safe script execution.
